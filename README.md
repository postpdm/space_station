# Goals

Big enterprise a looking for big enterprise soft. This is a bloat ERP's world, huge datacengers and colossal BI solutions.

Not all enterprise applications are necessarily complex and comprehensive.
Sometimes a large company needs simple solutions too. But when there are 100 of these simple solutions, managing them becomes increasingly difficult.

Our portal aggregates common functionality. It separates mature applications from small, experimental ones. It manages shared access, common functions and data flow.
